Game World Studio란?

대규모 언어모델과 생성형 AI를 결합해
게임의 세계관을 자동으로 구축하고,

왜 세계관이 중요한가?

  • 핵심 지적 자산으로서 작품의 가치 체계를 규정
  • 음악, 영상 등 파생 콘텐츠의 방향성 결정
  • 기획 단계에서 인물·지형·도시·풍경 등 수많은 요소를 일일이 설계해야 하는 부담
  • 창작자들이 느끼는 막막함을 줄여줄 도구의 필요성

시연 항목

  • 세계관 생성
  • NPC 프로필 생성 + 이미지 생성
  • NPC 대화 + 음성 + 비디오 생성
  • 3D 모델 생성
